# StudentEats – Campus Food Ordering App

StudentEats is an Android application designed specifically for students to easily discover and order food from nearby campus restaurants. The application focuses on simplicity, real-time updates, and a clean user interface suitable for student life.

## Features
- View available restaurants near campus
- Real-time restaurant listing using Firebase
- Simple and clean Android UI
- Admin panel to dynamically add restaurants
- Lightweight and fast application

## Technologies Used
- Java
- Android Studio
- Firebase Realtime Database
- XML Layouts
- RecyclerView

## System Architecture
The application follows a simple three-layer architecture:

Presentation Layer  
Handles user interface using Activities and XML layouts.

Logic Layer  
Java classes manage user interaction and data handling.

Data Layer  
Firebase Realtime Database stores restaurant data and updates it in real time.

## Project Structure
MainActivity – Displays restaurant list  
AdminActivity – Allows admin to add restaurants  
RecyclerView Adapter – Handles restaurant display

## Future Improvements
• Online payment integration
• Order history
• Restaurant ratings
• Location-based filtering
